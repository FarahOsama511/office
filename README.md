A Flutter application for managing orders between users and baristas efficiently and in real-time.
## Overview
Office App allows users to:

Create new orders with all details.
Edit orders before they are processed.
Delete orders before they are processed.
Orders are sent to the barista as a voice message.
Voice message contains order details.
Voice message contains the name of the user who placed the order.
The barista updates the order state as they progress.
Users receive real-time updates on the status of their orders.

## Features

Add orders.
Edit orders.
Delete orders.
Notifications for baristas when a new order is received.
Listen to the order via voice message to hear its details.
Real-time order status updates.

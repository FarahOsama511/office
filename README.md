A Flutter application for managing orders between users and baristas efficiently and in real-time.
## Overview
Office App allows users to:

- Add orders.  
- Edit orders.  
- Delete orders.  
- Notifications for baristas when a new order is received.  
- Listen to the order via voice message to hear its details.  
- Real-time order status update
